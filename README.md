## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Dev Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Customization
- **Theme**: Edit `tailwind.config.js` to change primary colors.
- **Content**: Update files in `src/data/` to change skills and projects without touching the UI code.

## Tech Stack
- React 18+
- Vite
- Tailwind CSS 3
